# AI Content Generator

A simple web-based tool that generates structured content posts based on user inputs like company name, topic, and tone. Built to explore how LLMs can be used in practical, workflow-driven applications.

## 🚀 Features
- Generates ready-to-use content (e.g., LinkedIn-style posts)
- Supports different tones (professional, casual)
- Structured output (hook, body, CTA)
- Simple input → prompt → output workflow

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- API: OpenRouter / OpenAI (LLM integration)

## 🧠 Purpose
This project was built to understand:
- How LLM APIs can be integrated into applications
- Prompt structuring for consistent outputs
- Basic AI workflow design (input → processing → output)
